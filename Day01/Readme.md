## What Is a Linux Distribution?

![Linux Distribution Diagram](https://upload.wikimedia.org/wikipedia/commons/1/1b/Linux_Distribution_Timeline.svg)

A Linux distribution (distro) is:
> A complete operating system built from the Linux kernel plus tools and applications.

---

## The Linux Kernel

![Linux Kernel Architecture](https://upload.wikimedia.org/wikipedia/commons/5/5a/Linux_kernel_map.png)

The kernel is the core of Linux:
- Manages hardware
- Handles memory
- Controls processes

---

## The Purpose of a Kernel

![Boot Process](https://upload.wikimedia.org/wikipedia/commons/6/63/Linux_boot_process.png)

### Boot Process
1. Firmware starts system
2. Bootloader loads kernel
3. Kernel initializes hardware
4. Services start
